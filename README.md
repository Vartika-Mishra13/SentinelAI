# SentinelAI – AI-Driven Intelligent API Gateway & Abuse Detection System

SentinelAI is a security-focused API Gateway that monitors API usage, detects abnormal traffic patterns, and automatically blocks abusive clients using intelligent anomaly detection.

## 🚀 Features
- JWT Authentication
- API Key Generation
- Sliding Window Rate Limiting
- Threat Level Detection (Low, Medium, High)
- AI-based Anomaly Detection (Burst Traffic, Traffic Spike, Continuous Abuse)
- Auto API Key Blocking System
- Interactive Dashboard with Charts
- Request History Logging
- Dark Mode UI
- Responsive Dashboard (Mobile Compatible)

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Charts: Chart.js
- Styling: CSS / Inline Styles

## 📊 Dashboard Features
- Threat Level Indicator
- API Usage Chart
- Request History
- AI Alerts Panel
- API Key Hide/Show
- Dark Mode Toggle
- Auto Block Countdown Timer

## ⚙️ How to Run Project

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

## 🔐 Project Purpose
This project simulates an intelligent API security gateway that protects APIs from abuse, DDoS-like attacks, and abnormal usage patterns using rate limiting and anomaly detection.
